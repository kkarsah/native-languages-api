üåç Native Languages API Documentation

Welcome to the Native Languages API! Our RESTful API provides authentic African language pronunciations, translations, and linguistic data for developers building educational applications, language learning platforms, and cultural preservation projects.

Quick Start

Get started with the Native Languages API in under 5 minutes:

  1. Get your free API key from the dashboard
  2. Make your first API call to /v1/languages
  3. Explore audio pronunciations with /v1/audio/{lang}/{word}
  4. Integrate into your application

üîê Authentication

The Native Languages API uses API keys for authentication. Include your API key in the X-API-Key header with every request.

API Key Usage 

curl "https://api.nativetongueapis.com/v1/languages" \
  -H "X-API-Key: your-api-key-here" \
  -H "Content-Type: application/json"
const response = await fetch('https://api.nativetongueapis.com/v1/languages', {
  headers: {
    'X-API-Key': 'your-api-key-here',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
import requests

headers = {
    'X-API-Key': 'your-api-key-here',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.nativetongueapis.com/v1/languages',
    headers=headers
)

data = response.json()
print(data)

    'X-API-Key: your-api-key-here',
    'Content-Type: application/json'
];

$response = curl_exec($ch);
$data = json_decode($response, true);

Get Languages

GET /v1/languages

Retrieves a list of all supported languages with their metadata including native names, ISO codes, speaker counts, and regional information.

Try it out

Response

Click "Send Request" to see the API response

Response Schema

Field Type Description
code string 3-letter language code (e.g., "swh")
name string English name of the language
native_name string Name in the native script
iso_code string ISO 639-1 language code
region string Primary geographical region
speakers_count integer Estimated number of speakers

Get Audio Pronunciation

GET /v1/audio/{language}/{word}

Retrieves audio pronunciation data for a specific word in the specified language. Returns metadata and audio file URL.

Try it out

Response

Click "Get Audio" to see the API response

Path Parameters

Parameter Type Required Description
language string Yes 3-letter language code (swh, yor, ibo, hau, amh)
word string Yes The word to get pronunciation for

Response Schema

Field Type Description
word string The requested word
language string Language code
translation string English translation
audio_url string Direct URL to audio file (MP3)
phonetic string IPA phonetic notation
duration_ms integer Audio duration in milliseconds

Error Codes

The Native Languages API uses conventional HTTP response codes to indicate the success or failure of API requests.

HTTP Status Codes

Code Status Description
200 OK Request succeeded
400 Bad Request Invalid request parameters
401 Unauthorized Invalid or missing API key
404 Not Found Resource not found
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Server error occurred

Error Response Format 

{
  "error": {

  }
}

import { NativeLanguagesAPI } from 'native-languages-api';

const api = new NativeLanguagesAPI({
  apiKey: 'your-api-key-here'
});

// Get all languages

// Get audio for a word
const audio = await api.audio.get('swh', 'jambo');
console.log(audio.audio_url);
